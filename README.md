# Dippin' Donuts

A donut shop landing page built using HTML and CSS.

This project was created after learning the basics of HTML and CSS as a practice exercise to improve layout building and understand how real web pages are structured. The design is based on a reference, but I added my own changes and styling decisions throughout the process.

---

## Tech Stack
- HTML
- CSS
- Basic SVG usage

---

## Features
- Hero section with branding and call-to-action
- Menu section with card-based layout
- Clean and simple UI design

---

## What I Learned
- CSS box model and spacing control
- Structuring a webpage using sections and divs
- When to use Flexbox vs Grid
- Difference between justify-content and align-items
- How to reuse components for cleaner structure
- Basic understanding of SVG usage
- Improving layout accuracy and alignment control
- Gained confidence building layouts without following a tutorial

---

## Process
I started by finding a simple and playful reference design to practice building a complete landing page.

While building this project, I focused on understanding layout structure, grouping elements properly, and improving my CSS decision-making. I also added small personal adjustments instead of strictly copying the reference.

---

## Overall Experience
This was a very useful learning project. Working without a step-by-step tutorial helped me understand how layouts are actually built and gave me more confidence in HTML and CSS.

I plan to continue building more projects to improve my frontend skills further.
